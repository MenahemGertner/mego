ארוחת בוקר

בכל בוקר הצוותים המסורים שלנו ישטפו, יחתכו ויכינו עבורך ארוחת בוקר טריה, טעימה ומזינה ללא מוצרים מעובדים. תוכל לבחירתך אף לשדרג את הארוחה לרכיבים אורגניים בלבד.

ארוחת צהרים

אנחנו מתמחים בהכנת ארוחות צהרים עשירות בטעמים אמיתיים כמו בבית, ללא שימוש במוצרים מעובדים. תוכל לבחירתך אף לשדרג את הארוחה לרכיבים אורגניים בלבד.
